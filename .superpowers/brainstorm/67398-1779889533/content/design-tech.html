2. Content Generation (từ framework slide R101)

💼 LinkedIn — Formal, 150–250 từ

Header (job title + company highlight) → Preview 2 câu hook → Main (bullet: công việc, yêu cầu, quyền lợi) → CTA (link apply hoặc DM). Tone: chuyên nghiệp, dùng emoji chừng mực.

📘 Facebook Group — Friendly, 80–150 từ

Hook 1–2 câu đầu mạnh (3s rule — chỉ show 125 ký tự), tone thân thiện như người thật đang seeding, ít bullet hơn, emoji nhiều hơn, CTA rõ.

🧵 Threads — Punchy, max 500 ký tự

1 câu hook + 3 bullet siêu ngắn + CTA. Không dùng ảnh. Đủ để ai đó stop scroll và DM.

📄 TopCV/VietnamWorks — Full JD format

Re-export JD đã có sẵn theo format chuẩn job board. Jane chỉ reformat, không generate lại. Content-only, user tự paste.

Tất cả content editable trước khi post. API: POST /api/post-job/generate — nhận JD + job_type → trả về object {linkedin, facebook, threads, topcv}.

3. OAuth Integration

LinkedIn OAuth 2.0

  • Scope: w_member_social
  • Post as status/share (không cần Talent partner)
  • Token lưu Supabase (encrypted), refresh tự động

Facebook Graph API

  • Scope: pages_manage_posts + groups_access_member_info
  • Rec chọn Page/Group nào muốn post
  • Token long-lived (60 ngày), auto-refresh

🧵 Threads API (Meta)

Dùng Threads Publishing API (released 2024). Scope: threads_basic + threads_content_publish. Cùng app Facebook → add thêm scope, không cần app riêng.

4. Data Model

-- Bảng mới: post_campaigns
post_campaigns {
  id uuid PK
  jd_history_id uuid → jd_history.id
  channel text -- 'linkedin'|'facebook'|'threads'|'topcv'
  content text -- generated content (editable)
  status text -- 'draft'|'posted'|'failed'
  platform_post_id text -- ID trả về từ platform
  posted_at timestamptz
  created_at timestamptz
}

-- Bảng mới: connected_accounts
connected_accounts {
  id uuid PK
  user_id text -- Clerk user ID
  platform text -- 'linkedin'|'facebook'|'threads'
  access_token text -- encrypted
  refresh_token text
  token_expires_at timestamptz
  platform_user_id text
  platform_user_name text -- hiển thị "Đang post với @..."
  created_at timestamptz
}

5. API Routes mới

POST
/api/post-job/generate — nhận jd_history_id → classify job type → generate content cho 4 kênh
POST
/api/post-job/publish — nhận campaign_id + channel → post lên platform → update status
GET
/api/auth/[platform]/connect — redirect sang OAuth flow (linkedin/facebook/threads)
GET
/api/auth/[platform]/callback — nhận code → exchange token → lưu connected_accounts
GET
/api/post-job/campaigns?jd_id= — lấy lịch sử đăng cho 1 JD
DELETE
/api/auth/[platform]/disconnect — xóa token, ngắt kết nối account
