3 cách build feature này

Chọn approach phù hợp với timeline và độ phức tạp bạn chấp nhận được

A

Content Generator Only (không OAuth)

Build time: ~3–5 ngày. Jane phân tích JD → gợi ý kênh phù hợp → generate sẵn nội dung cho LinkedIn / Facebook Group / Threads / Job boards → rec copy-paste tự đăng. Không cần connect account. Nhanh ship, zero risk về API.

Nhanh nhất Rec vẫn phải tự đăng
B ✦

Content Generator + OAuth Phased (Recommended)

V1 (~1 tuần): Content gen + LinkedIn OAuth post-as-status + Facebook Page/Group OAuth. V2 (~sau đó): Threads API + Job boards (TopCV free 3 posts). LinkedIn post dưới dạng status bình thường — không cần LinkedIn Talent partner, rec vẫn dùng được ngay.

Ship được sớm Direct post cho kênh quan trọng nhất LinkedIn = status, không phải job listing
C

Full Platform Suite ngay từ đầu

Build time: 3–4 tuần. LinkedIn + Facebook + Threads + TopCV browser automation (Playwright) cùng lúc. Token management, error handling, re-auth flow cho tất cả. Xịn nhất nhưng risk cao — mỗi platform có thể break độc lập.

Đầy đủ nhất Ship chậm Browser automation fragile

💡 Note về LinkedIn API

LinkedIn Job Posting API chính thức cần partner program (rất khó xin). Nhưng Share API (post status bình thường) thì dễ — rec connect LinkedIn → Jane post "đang tuyển [vị trí]..." như một status trên feed của họ. Thực tế đây là cách rec VN hay làm nhất rồi.
