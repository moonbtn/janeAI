Platform nào quan trọng nhất để post trực tiếp?

Mỗi platform build OAuth riêng — chọn theo mức độ ưu tiên thực tế của rec

💼
LinkedIn API có sẵn ⚠️ Job API cần partner

Post personal update dễ (OAuth v2). Post job listing chính thức cần LinkedIn Talent partner — khó xin. → Có thể workaround bằng cách post như một status/article thay vì job listing.

📘
Facebook Page / Group Graph API ổn định

Post lên Page công ty hoặc Group mình quản lý — Graph API dễ dùng, không cần partner. Rec chỉ cần login Facebook một lần là Jane post được.

📄
TopCV / VietnamWorks / ITviec Không có public API

Job boards VN chưa có API công khai. Cách khả thi nhất: browser automation (Playwright) — post qua form thật. Fragile hơn, nhưng làm được.

💬
Zalo / Telegram Groups Bot API

Zalo OA API có, Telegram Bot API dễ dùng. Cần rec là admin group. Phù hợp cho seeding vào các nhóm nghề.
